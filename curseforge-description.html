HyperFactions is a complete faction management system for Hytale servers. Create factions, claim territory, forge alliances, and wage war — all through an intuitive GUI or commands.

Early Access: Actively developing. Report bugs on GitHub!

 

 

KEY FEATURES

Factions & Territory

  • Create factions with custom names, colors, and tags
  • Chunk-based territory claiming with visual map
  • Power system that rewards active play
  • Three-tier hierarchy: Leader, Officer, Member
  • Diplomacy system with allies, enemies, and truces

Protection System

  • 20+ configurable zone flags (PvP, building, containers, mob spawning, etc.)
  • Admin SafeZones and WarZones
  • Per-faction territory permissions
  • Combat tagging system

Admin Tools

  • Full GUI administration panel
  • Automatic backup system with smart rotation
  • Debug mode for troubleshooting
  • Bypass mode for investigations

 

 

OPTIONAL DEPENDENCIES

Strongly Recommended:

  • HyperPerms — Permission-based limits for claims, power, and features
  • VaultUnlocked — Chat, economy, and permission compatibility with other mods

Recommended for Full Protection:

  • Hyxin — Mixin loader (enables protection hooks)
  • OrbisGuard-Mixins — F-key pickup protection, keep inventory, invincible items

Optional:

  • OrbisGuard — Region protection integration (work in progress)

 

 

INSTALLATION

Basic Install:

  1. Drop HyperFactions.jar in your mods/ folder
  2. Start your server
  3. Type /f to open the GUI

For Full Protection (Hyxin + OrbisGuard-Mixins):

  1. Create an earlyplugins/ folder in your server directory
  2. Place Hyxin and OrbisGuard-Mixins in earlyplugins/ (NOT mods/)
  3. Edit your server start script to add the --accept-early-plugins flag:

Linux (start.sh):

DEFAULT_ARGS="--accept-early-plugins --assets ../Assets.zip --backup --backup-dir backups --backup-frequency 30"

Windows (start.bat):

set DEFAULT_ARGS=--accept-early-plugins --assets ../Assets.zip --backup --backup-dir backups --backup-frequency 30

HyperFactions works without these, but some zone protections (F-key pickup blocking, keep inventory, invincible items) require the mixin system.

 

 

QUICK START

1. Grant permissions (with HyperPerms):

/hp group setperm default hyperfactions.use
/hp group setperm default hyperfactions.faction.*
/hp group setperm default hyperfactions.member.*
/hp group setperm default hyperfactions.territory.*
/hp group setperm default hyperfactions.teleport.*
/hp group setperm default hyperfactions.relation.*
/hp group setperm default hyperfactions.chat.*
/hp group setperm default hyperfactions.info.*

2. Start playing:

Players: /f create MyFaction or just /f for the GUI

Admins: /f admin for the admin panel
